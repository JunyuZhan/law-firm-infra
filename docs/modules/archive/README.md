# 档案管理模块说明

## 功能说明

档案管理模块提供了档案的分类、存储和借阅管理功能，主要包括：

1. 档案分类管理
   - 分类创建与编辑
   - 分类树管理
   - 分类编码管理
   - 分类权限控制

2. 档案基本信息管理
   - 档案登记
   - 档案编目
   - 档案查询
   - 档案统计

3. 档案借阅管理
   - 借阅申请
   - 借阅审批
   - 借阅归还
   - 超期提醒

4. 档案安全管理
   - 密级管理
   - 访问控制
   - 操作日志
   - 安全审计
